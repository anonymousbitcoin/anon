Detailed Description

256-bit opaque blob.

Note
This type is called uint256 for historical reasons only. It is an opaque blob of 256 bits and has no integer operations. Use arith_uint256 if those are required.

◆ GetCheapHash()

uint64_t uint256::GetCheapHash ( ) const
inline

A cheap hash function that just returns 64 bits from the result, it can be used when the contents are considered uniformly random. It is not appropriate when the value can easily be influenced from outside as e.g. a network adversary could provide values to trigger worst-case behavior.

Note
The result of this function is not stable between little and big endian.
