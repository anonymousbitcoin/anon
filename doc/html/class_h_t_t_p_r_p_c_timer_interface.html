Factory function for timers. RPC will call the function to create a timer that will call func in millis milliseconds.

Note
As the RPC mechanism is backend-neutral, it can use different implementations of timers. This is needed to cope with the case in which there is no HTTP server, but only GUI RPC console, and to break the dependency of rpcserver on httprpc.
